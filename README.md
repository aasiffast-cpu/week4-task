# Flask Blog

A simple blog website built with Flask.

## Features
Add, edit, delete posts
View all posts
Validation & error handling
SQLite database
Bootstrap styling

## Usage
1. Clone repo
2. Install requirements: `pip install -r requirements.txt`
3. Run app: `python run.py`
4. Open in browser: `http://127.0.0.1:5000`
